# Módulo 01 — Prática

Exercícios práticos da apostila. Rode cada script com `node <arquivo>`.

Para os exercícios 4 e 5 (axios e chalk), instale as dependências antes:

```bash
npm install
```

## Mapa dos exercícios

| Exercício da apostila | Arquivo |
| --- | --- |
| 1. Verificar a versão do Node | `01-bem-vindo.js` (imprime `process.version`) |
| 2. Exibir "Bem-vindo ao Node.js!" | `01-bem-vindo.js` |
| 3. Mensagem com atraso de 3 segundos | `02-atraso.js` |
| 4. GET a uma API com axios | `03-axios-get.js` |
| 5. Mensagem de erro em vermelho com chalk | `04-chalk-erro.js` |
| 6. Ler `texto.txt` e exibir no console | `05-ler-arquivo.js` + `texto.txt` |
| 7. Somar dois números com callback | `06-soma-callback.js` |
| 8. Dependência de desenvolvimento | `package.json` (`devDependencies`: nodemon) |
| 9. Criar e ler um JSON de usuário | `07-ler-json.js` + `usuario.json` |
| 10. Exibir data e hora formatadas | `08-data-hora.js` |
