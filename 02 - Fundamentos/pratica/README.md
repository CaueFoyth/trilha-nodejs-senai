# Módulo 02 — Prática

Todos os exercícios usam apenas módulos nativos do Node (sem `npm install`). Rode com `node <arquivo>`.

Os dois servidores HTTP (`01` e `09`) ficam escutando na porta 3000 — pare com `Ctrl+C`.

## Mapa dos exercícios

| Exercício da apostila | Arquivo |
| --- | --- |
| 1. Servidor HTTP com "Bem-vindo ao Node.js!" | `01-servidor-http.js` |
| 2. Criar e ler um arquivo com fs | `02-fs-arquivo.js` |
| 3. Stream para processar um arquivo grande | `03-stream-leitura.js` |
| 4. Buffers: string → binário | `04-buffers.js` |
| 5. Módulo CommonJS exportado e importado | `calculadora.js` + `05-usar-modulo.js` |
| 6. Configurar e ler um JSON com fs | `06-json.js` + `config.json` |
| 7. try/catch em arquivo inexistente | `07-try-catch.js` |
| 8. Stream Duplex/Transform modificando dados | `08-transform.js` |
| 9. Servidor HTTP com resposta por rota | `09-servidor-rotas.js` |
| 10. Debug com `--inspect` | `10-debug.js` |
