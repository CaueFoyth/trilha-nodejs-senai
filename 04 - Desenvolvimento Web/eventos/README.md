# Módulo 04 — Atividades práticas de EventEmitter

Scripts sem dependências. Rode com `node <arquivo>`.

| Atividade da apostila | Arquivo |
| --- | --- |
| Criação de eventos simples (`hello`) | `01-hello.js` |
| Sistema de log (info, warn, error) | `02-log.js` |
| Simulação de chat (`userJoined`, `message`) | `03-chat.js` |
| Evento com múltiplos listeners (arquivo + console) | `04-multiplos-listeners.js` |
| Gerenciamento de erros com evento `error` | `05-erros.js` |
